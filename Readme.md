# 🏨 LUXURY HOTEL WEBSITE

This is a beginner-friendly frontend hotel website project built using HTML, CSS, and basic responsive design concepts.

## Website Type

Hotel / Hospitality Website

## Technologies Used

* HTML5
* CSS3
* JavaScript (basic interactions, if included)
* GitHub (version control)
* Netlify (deployment)

## Pages Included

* Home (`index.html`)
* About (`about.html`)
* Rooms (`rooms.html`)
* Services (`services.html`)
* Gallery (`gallery.html`)
* Booking (`booking.html`)
* Contact (`contact.html`)

## Features

* Responsive hotel website design for mobile, tablet, and desktop
* Sticky navigation menu
* Attractive hero section with hotel introduction
* Multi-page structure for better organization
* Room booking form
* Services section with hotel facilities
* Rooms section showcasing room categories
* Gallery section displaying hotel interiors and amenities
* Testimonials section for customer reviews
* Contact page with hotel details and map location
* Smooth hover animations and transitions
* Fast-loading static website

## UI Design

* Primary Color: Gold (#c59d5f)
* Secondary Color: Dark Red (#8B0000)
* Font Style: Poppins
* Layout: Header Navigation + Main Content Area
* Style: Clean, modern, luxury hotel-themed UI

## How to Run

1. Download or clone the project
2. Open the folder in any code editor (VS Code recommended)
3. Open index.html in a browser
4. Use the navigation menu to browse all pages

## Netlify Deployment Steps

1. Create a GitHub repository
2. Upload all project files
3. Login to Netlify
4. Click “Add new site” → “Import an existing project”
5. Connect your GitHub repository
6. Click Deploy Site
